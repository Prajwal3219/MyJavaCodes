# Java practice programs

This repository contains a set of small Java programs used for practice exercises. Files are single-class programs demonstrating basics like loops, conditionals, and simple algorithms.

Files include:

- `Add.java` — simple addition program
- `EvenOdd.java` — checks even/odd
- `fact.java` — factorial calculation
- `Fibbo.java` — Fibonacci sequence
- `LargetsNo.java` — find largest number
- `NumberRev.java` — reverse a number
- `Pallindrom.java` — palindrome check
- `Prime.java` — prime number test
- `RevrseArray.java` — reverse an array
- `Star.java` — prints star patterns
